CÓMO ABRIRLO EN VISUAL STUDIO CODE
----------------------------------
Opción A — La más fácil (doble clic):
  1. Descomprime esta carpeta donde quieras.
  2. Haz doble clic en "index.html".
  3. Se abrirá en tu navegador. ¡Listo!

Opción B — Con Live Server (recomendado):
  1. Abre la CARPETA "leonardo-portfolio" en VS Code
     (Archivo → Abrir carpeta).
  2. Instala la extensión "Live Server" si no la tienes.
  3. Haz clic derecho sobre "index.html"
     y elige "Open with Live Server".

NOTAS
-----
• No requiere instalación de nada (no Node, no React, no build).
• Las fuentes Poppins y JetBrains Mono se cargan desde Google Fonts
  (necesitas internet la primera vez).
• El mapa de la sección de contacto usa Google Maps (también requiere internet).
• Funciona en Chrome, Edge, Firefox y Safari modernos.
